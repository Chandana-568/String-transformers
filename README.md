String Transformer

live link:
https://bmi-calculator-568.netlify.app/

A simple and interactive web tool to transform and manipulate text in real-time. This project demonstrates the use of **JavaScript DOM manipulation, event handling, and dynamic UI updates** in a clean and responsive interface.

>Features

- Convert text to **uppercase** or **lowercase**  
- Count characters and words  
- Remove extra spaces for cleaner formatting  
- Real-time text transformation as the user types  
- Responsive and user-friendly design
